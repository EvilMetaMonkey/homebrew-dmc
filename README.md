# EvilMetaMonkey/homebrew-dmc

Homebrew tap for [dmc](https://github.com/EvilMetaMonkey/dmc) — a local shadow-history version control tool for AI-assisted coding.

## Installation

```bash
brew tap EvilMetaMonkey/dmc
brew install dmc
```

## Updating

```bash
brew update
brew upgrade dmc
```

## Uninstalling

```bash
brew uninstall dmc
brew untap EvilMetaMonkey/dmc
```

## Supported Platforms

- macOS Intel (x86_64)
- macOS Apple Silicon (arm64)
- Linux (x86_64)

Windows binaries are available from the [GitHub Releases](https://github.com/EvilMetaMonkey/dmc/releases) page.
